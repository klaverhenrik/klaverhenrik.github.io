Transfer learning with Kears and MobileNet

Description

This demo uses a customized Convolutional Neural Network (CNN) called MobileNet to recognize objects in images. The CNN has been pretrained on ImageNet and subsequently fine-tuned using Keras to recognize objects of my personal choice (in this case cats, dogs, and horses).

You can easily retrain the CNN to recognize objects of your choice using Google Colab. I have provided a template with detailed instuctions, which you can find here.
